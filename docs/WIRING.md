# Подключение

## UART

Используемые параметры:

```
Скорость: 9600 бод
Данные: 8 бит
Чётность: EVEN
Стоп-биты: 1
```

Типовая распиновка:

```
ESP32-C3 GPIO3 (TX)  -> RX кондиционера
ESP32-C3 GPIO1 (RX)  <- TX кондиционера через согласование уровня
ESP32-C3 GND         <-> GND кондиционера
```

## Согласование линии TX кондиционера -> RX ESP32

Если на TX кондиционера измеряется около 5 В, используйте резистивный делитель:

```
TX кондиционера ---- 10 кОм ----+---- RX ESP32-C3
                                |
                               20 кОм
                                |
                               GND
```

Расчёт при входном напряжении 4,9 В:

```
4,9 × 20 / (10 + 20) ≈ 3,27 В
```

## Линия TX ESP32 -> RX кондиционера

Если кондиционер стабильно принимает уровень 3,3 В, GPIO TX можно подключить напрямую:

```
TX ESP32-C3 -> RX кондиционера
```

## Питание

Подключение питания зависит от конкретной платы ESP и разъёма кондиционера.

Не подавайте питание одновременно несколькими способами без понимания схемы платы. Для прошивки по USB безопаснее отключать питание от кондиционера.

## Важно

- земля должна быть общей;
- сначала измерьте реальные напряжения мультиметром;
- не рассчитывайте на 5-вольтовую толерантность GPIO ESP32-C3;
- неизвестную распиновку определяйте до подключения ESP.
