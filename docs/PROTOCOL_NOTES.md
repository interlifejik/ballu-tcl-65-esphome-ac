# Технические заметки по протоколу

## Формат ответа

Проверенная ревизия отвечает пакетом:

```
Header: 5 байт
Body:   0x3B = 59 байт
XOR:    1 байт
Total:  65 байт
```

## Запрос состояния

```
BB 00 01 04 02 01 00 BD
```

## Температура

В управляющей команде целевая температура кодируется полным байтом:

```
byte_9 = 0x6F - target_temperature
```

Пример для 16 °C:

```
0x6F - 16 = 0x5F
```

## Жалюзи

Компонент поддерживает:

- движение по полному диапазону;
- частичные диапазоны;
- фиксированные позиции;
- вертикальную и горизонтальную оси.

Расширенные позиции представлены отдельными `select`-сущностями, поскольку стандартный API `climate` Home Assistant поддерживает только `OFF`, `VERTICAL`, `HORIZONTAL` и `BOTH`.

## Звук и дисплей

В управляющем байте используются отдельные флаги:

- звуковое подтверждение команды;
- дисплей внутреннего блока.

Переключатели сохраняют выбранное состояние и применяют его ко всем последующим командам ESP32.
